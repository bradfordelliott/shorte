1. An Introduction

It is assumed that a heading tag is implictly followed by an @text tag. This creates a text block which is the most common tag.
You can also start new text blocks with the @text tag.

1.1. Tables

The @table tag creats a table:
An Example Table
Heading 1
Heading 2
SubHeading 1 that Spans
Column 1
Column 2

1.2. Lists

You can create ordered lists using the ol tag:
  1. one
    1. two
      1. three
  2. four
    1. five
      1. six
Or unorded lists using the ul tag:
  • one
    • two
      • three
  • four
    • five
      • six
